LaTeX

Equation Editor - build equations from drop-down menus and preview typesetting before copying LaTeX commands into document MiKTeX - typesetting base; includes bare-bones TeXworks editor; install new packages from Maintenance\Package Manager TeXstudio - TeX editor with greater functionality; includes pdf preview, symbol menus, and wizards for tables, page setup, and more Wikibook - resource for LaTeX markup language; includes a special section on use in Linguistics JabRef reference manager - organizes and displays references in .bib files; nice features include file linking (e.g. pdf of an article linked to its reference) Google Scholar - search for a reference and click "cite" in bottom right corner of search result; next window has an option: "Import into BibTeX"

Proudly powered by Pelican, which takes great advantage of Python. Theme is using cebong.
